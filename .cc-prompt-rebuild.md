## 重构规则

### ✅ 允许
- 自由修改所有 CSS property 值
- 自由添加/删除/重命名 CSS class（这次推倒重来，v4 strict mode 取消）
- 自由修改 TSX 结构和 className 引用
- 添加新的 CSS 变量
- 调整组件 DOM 结构以匹配设计稿

### 🚫 禁止
- **禁止 inline style** — 所有样式必须通过 CSS class 或 CSS 变量
- **禁止 `width: 100%` 在 flex 子元素上** — 会阻止 max-width + margin auto 居中
- **禁止猜测数值** — 所有间距/颜色/字号从设计稿 HTML 或 DESIGN.md 提取
- **禁止改变功能逻辑** — 只改 UI 层，不改 store/type/API 逻辑
- **禁止删除功能组件** — 每个组件保留其功能，只重做样式

### ⚠️ JS ↔ CSS 同步
- `InputArea.tsx` 里的 auto-resize: `Math.max(el.scrollHeight, N)` 中的 N 必须和 CSS `min-height` 一致（当前是 100）

### 侧边栏规格
- 宽度：48px，纯图标，无文字
- Logo：32px 圆角 gradient orange + 白色 terminal 图标
- 导航：居中图标，hover 有背景
- 活跃项：橙色左边框 + 橙色背景
- 底部：主题切换 + 设置图标

### 消息布局规格
- AI 消息：LEFT 对齐，头像在左
- 用户消息：RIGHT 对齐，头像在右
- 用户气泡：`border-radius: 24px 24px 24px 4px`
- 消息区居中：`max-width: 768px; margin: 0 auto`

### 输入框规格
- `min-height: 100px`（JS auto-resize 最小值同步）
- `max-height: 200px`
- `border-radius: 16px`
- `max-width: 768px; margin: 0 auto`（居中）
- 发送按钮：brand-orange，右下角
